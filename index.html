dObfuscator

What is this?

This is an simple mirror for Anti-Malware programs, in the case where the malware prevents the download, this could be used to bypass that.

Every download has a randomized file name.

For example:

AdwCleaner
Version: v5.118
SHA-256: ef525efe477662d4f3e783af7f0d103e0ebf81c76af1f6f48f506953932bef97
